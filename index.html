Calculator

Description


Hi Everyone. This is a simple calculator which I have made using DOM. Here you can perform some basic arithematic operations such as addition, subtraction, division, multiplication and modulus. For building this UI I have used DOM for creating HTML elements, CSS and Bootstrap for styling. This UI is responsive for all media screens. I have included some additional buttons like C for clearing and D for deleting or removing the last digit from the display-input screen. In case if you performs an invalid operation then the screen will show you an alert message and clears the invalid operation from the display-input screen.
